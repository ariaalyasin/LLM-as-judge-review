# LLM-as-a-Judge Literature Review

Welcome to our university project repository! 

## Collaboration Protocol

To keep our work organized across the different phases, please follow these simple rules:

1. **Work in the Correct Phase Folder:** We are currently on `Phase_1`. Everything related to this phase goes in there.
2. **Articles:** Place all downloaded PDF papers in `Phase_1/articles/`. Please use clear filenames (e.g., `Year_Author_Title.pdf`).
3. **Summaries:** Write your notes and paper drafts in `Phase_1/summaries/`. Markdown (`.md`) files are preferred for simple reading.
4. **Presentations:** All slides will be made using **LaTeX**. Put your `.tex` files in `Phase_1/presentation/` and place all related figures/graphics in `Phase_1/presentation/images/`.
5. **Git Workflow:** Always `git pull` before making changes to avoid conflicts, and try to write descriptive commit messages when you push your work.
